# Vanilla-Addons
建立这个项目只是用来存放自己在60级版本魔兽世界所使用的一些插件，也为了记录插件的更新情况。 
大部分的插件来自[60addons.com](http://60addons.com/ "60addons.com")，感谢站长Qcat的无私奉献！

3. 德鲁伊、暗牧自动解除形态需要，增加了`Automaton`插件，同时移除了`AutoAttack`插件。
